    • Define what it mean for two paths $\alpha, \beta : [0, 1] \to X$, with the same start points and the same end points, to be homotopic. Show that this is an equivalence relation.

      See what is the fundamental group?.

      Let $\alpha, \beta : [0, 1] \to X$ such that $\alpha(0) = \beta(0) = x_0 \in X$ be loops in $X$. We say that $\alpha$ is homotopic to $\beta$ if there exists a function $H : [0, 1] \times [0, 1] \to X$ such that the following hold
      1. H(0, \cdot) = \alpha
      2. H(1, \cdot) = \beta
      3. $H$ is continuous
      We sketch the proof that this is an equivalence relation

      • Reflexivity

        Simply consider the identity homotopy $H(t, x) = \alpha(x)$.

      • Symmetry

        Let $H$ be the homotopy between $\alpha$ and $\beta$. Then $\widetilde{H} = H(1 - t, x)$ is a homotopy between $\beta$ and $\alpha$ since $1 - t$ is a continuous function.

      • Transitivity

        Let $H_1$ be the homotopy between $\alpha$ and $\beta$, and $H_2$ the homotopy between $\beta$ and $\gamma$. Then $\widetilde{H}(x, t) = H_1$ for $t \in [0, 1/2]$ and $\widetilde{H}(x, t) = H_2$ for $t \in (1/2, 1]$ is a homotopy between $\alpha$ and $\gamma$.

    • Define what it means for a space to be simply connected.

      Simply connected means the space has no holes. See what is the fundamental group?.
      If $X$ is a path connected space, we say that $X$ is simply connected if \[ \pi_1(X, x_0) = 1. \] That is, the fundamental group of $X$ is trivial.

    • Give a complete statement of the Seifert-van Kampen theorem relating the fundamental groups of $U, V, U \cup V$, and $U \cap V$, including all necessary assumptions.

      See What does Van Kampen's Theorem Really Say
      Let $X = U \cup V$ be a path-connected space with $U, V$ open path-connected subsets such that $U \cap V$ is path connected. Then the fundamental group of $X$ is given by \[ \pi_1(X, x_0) = \pi_1(U, u_0) * \pi_1(V, v_0) / N \] where $*$ denotes the free product and $N$ is generated by the set \[ \{ \pi_1(i_U)(\gamma)\pi_1(i_V)(\gamma)^{-1} \mid \gamma in \pi_1(U \cap V, y_0) \}. \] where $i_U, i_V$ are the inclusion maps from $U, V$ to $X$, respectively.

    • State the classification theorem relating (connected) covering spaces of a (connected) space $X$ to the fundamental group of $X$ (you may take as given the standard assumptoins that $X$ is locally path-connected and locally simply connected.)

      See how to classify the symmetries of an object.
      There are two statements of the theorem. We offer the simpler one below.
      We say a space $X$ is path connected if between any two points $a, b$ there exists a path $\gamma$ such that $\gamma(0) = a$ and $\gamma(1) = b$.

      We say a space $X$ is locally path connected if for point $p \in X$ and neighborhood $U$ of $p$, there exists a smaller neighborhood $V$ such that $V$ is path connected.

      Note: due to annoying naming conventions, path connected does not imply locally path connected. Consider the comb space. Let $K = \{ 1/ n \mid n \in \Nn \}$. Then \[ X = \{ 0 \} \times [0, 1] \cup [0, 1] \times \{0 \} \cup K \times [0, 1] \] with the subspace topology inherited from $\R^2$. The space is path connected, but is not locally path connected on the vertical line at the origin.

      Statement 1
      Let $X$ be a path-connected, locally path-connected, and semi-locally path connected. There is a bijection between basepoint preserving isomorphism classes of connected covering spaces and subgroups of $\pi_1(X, x_0)$.
      Statement 2
      Let $X$ be a path-connected, locally path-connected, and semi-locally path connected. There is a bijection between isomorphism classes (not necessarily basepoint preserving) of covering spaces and conjugacy classes of $\pi_1(X, x_0)$.
      Hence, removing the assumption that the isomorphism between two covering spaces preserves the basepoint, changes the bijection to conjugacy classes.

    • Calculate the homology groups of the torus $S^1 \times S^1$.

      See what is homology?. Build the torus from one $0$-simplex, three $1$-simplices, and two $2$-simplices as follows The first homology group $H_0 = \Z$ since the space is connected. Since the torus is path connected, then $H_1 = \pi_1 = \Z \times \Z = \Z \oplus \Z$. Finally, we have $H_2 = \langle \lambda_1 - \lambda_2 \rangle / \langle 0 \rangle = \Z$. Hence, the homology is \[ H_k(T) = \begin{cases} \Z & \textrm{ if } k = 0, 2 \cr \Z \oplus \Z & \textrm { if } k = 1 \end{cases} \]
      We present arguments varying in computation and abstraction.

      Computation from definitions alone (simple)

      From first definitions alone we can compute the homology group. We start by computing the kernels and images of the boundary maps. \[ \begin{aligned} \textrm{ker } \partial_0 &= \Z \cr \textrm{img } \partial_0 &= 0 \cr \textrm{ker } \partial_1 &= \langle a, b, c \rangle \cr \textrm{img } \partial_1 &= \langle 0 \rangle \cr \textrm{ker } \partial_2 &= \langle \lambda_1 - \lambda_2 \rangle \cr \textrm{img } \partial_2 &= \langle a + b - c\rangle \cr \textrm{ker } \partial_3 &= 0 \cr \textrm{img } \partial_3 &= 0\rangle \end{aligned} \] Note that $\textrm{ker } \partial_2$ because \[ \partial_2(p \lambda_1 + q \lambda_2) = (p + q)(a + b - c) = 0. \] Since $a + b - c$ is a element of the basis $\{ a, b, a + b - c \}$ for $C_1$, then $p + q = 0$ so $p = -q$. With this information we can compute the homology groups as follows \[ \begin{aligned} H_0 &= \frac{\Z}{0} = \Z \cr H_1 &= \frac{\textrm{ker } \partial_1}{\textrm{img } \partial_2} = \frac{\langle a, b, c \rangle}{\langle a + b - c \rangle} = \Z \oplus \Z \oplus \Z / \Z = \Z \oplus \Z \cr H_2 &= \frac{\textrm{ker } \partial_2}{\textrm{img } \partial_3} = \frac{\langle \lambda_1 - \lambda_2 \rangle}{0} = \Z \end{aligned} \] Thus, the homology of the torus is \[ H_k(T) = \begin{cases} \Z & \textrm{ if } k = 0, 2 \cr \Z \oplus \Z & \textrm { if } k = 1 \end{cases} \]

      Computation with two tools (simpler)

      First, note that $T^2$ is a connected space. A general fact implies that $H_0(T^2) = \Z$. Then note that $\pi_1(T^2, x_0) = \Z \oplus \Z$. Another general fact tells us that for any path-connected topological space $X$, we have $H_1(X) = \pi_1(X, x_0) / \langle aba^{-1}b^{-1} \rangle$ (i.e. $H_1(X)$ is the abelianization of $\pi_1(X, x_0))$. Thus $H_1(T^2) = \pi_1(T^2, x_0) = \Z \oplus \Z$.

      For $k > 2$, note that $H_k(T^2) = 0$ since the highest dimensional complex is a $2$-complex, so the higher chain groups vanish. By definition, this implies that \[ H_2(T^2) = \frac{\textrm{ker } \partial_2}{\textrm{im } \partial 3} =\frac{\textrm{ker } \partial_2}{0} = \textrm{ker } \partial_2 \] Notice that the two simplices $\lambda_1, \lambda_2$ used in the complex for $T^2$ share a common boundary, so they both map to the same $1$-cycle under $\partial_2$. In general, if two vectors map to the same thing under a linear map, their difference is in the kernel. Hence $\textrm{ker }\partial_2 = \langle \lambda_1 - \lambda_2 \rangle$.

      As in the previous computation, we have thus concluded that \[ H_k(T) = \begin{cases} \Z & \textrm{ if } k = 0, 2 \cr \Z \oplus \Z & \textrm { if } k = 1 \end{cases} \]

      Computation with three tools (simplest)

      As above, since $T^2$ is connected and $\pi_1(T^2, x_0) = \Z \oplus \Z$, we have that $H_0(T^2) = \Z$ and $H_1(T^2) = \pi_1(T^2, x_0) = \Z \oplus \Z$. In general, if $X$ is a smooth, compact, connected, orientable $n$-manifold, without boundary, then the top homology is $H_n(X) = \Z$. So for the torus, which satisfies these hypotheses, we have $H_2(T^2) = \Z$. The higher homology groups vanish because the higher chain groups vanish.

    • For $n \geq 1$, let $i$ be the inclusion $S^{n -1} \to D^n$. Show that there is no continuous map $r : D^n \to S^{n - 1}$ such that $r \circ i = id$. (You may take the homology of these spaces as known, so long as you state them correctly).

      If such a map existed, then $r$ would be a retract from $D^n$ to $S^{n-1}$ which is unlikely. In the special case of $D^2$ to $S^1$, the problem is clear. The disk $D^2$ is simply connected while $S^1$ is not. The retraction includes $\pi_1(S^1) = \Z$ into $\pi_1(D^2) = 1$ which is impossible. The problem is that we cannot deform the space $D^2$ to $S^1$ without tearing it since $S^1$ has a hole while $D^2$ does not.
      Note: for the purpose of this proof, assume that all homology is reduced homology. That is, points have trivial homology (see Hatcher p. 110).
      We call a continuous map $r : X \to A$ where $A \subset X$ a retract if $r(X) = A$ and $r(a) = a$ for $a \in A$. Equivalently, $r$ is a retract if $r \circ i = id$ where $i : A \to X$ is the inclusion map.

      Assume towards a contradiction that there is a retraction $r : D^n \to S^{n - 1}$. Then we know that $r \circ i = \textrm{id}$ induces homomorphisms \[ (r \circ i)^* = r^* \circ i^* = \textrm{id}_{H_k(S^n)} \] so we have the following exact chain sequence If $n > 1$, then the composition $r^* \circ i^*$ cannot be equal to the identity because it factors through $H_{n - 1}(D^n) = 0$ with $H_{n - 1}(S^{n - 1}) = \Z$. If $n = 1$, then $r^* \circ i^*$ cannot be equal to the identity because it factors through $H_0(D^1) = \Z$ and $H_0(S^0) = \Z \plus \Z$. This implies that $r^*$ is not surjective, so the composition $r^* \circ i^*$ cannot be the identity.

      Note: one can prove this in one line using reduced homology.

    • For $n > 0$, define the degree of a continuous map $S^n \to S^n$.

      The general definition (stated below) generalizes the simple idea that the degree of $f$ measures how many times it winds a loop of $S^1$ around $S^1$.

      For example, the degree of the identity is $1$, the degree of $z \mapsto z^2$ is two, the degree of a reflection is $-1$, and the degree of the antipodal map is $1$ (a rotation by $pi$, so homotopic to the identity).

      Most importantly, the maps $f$ and $g$ are homotopic if and only if they have the same degree. Hence, we can see that the fundamental group of $S^1$ is indeed $\Z$. This construction is earlier than the fundamental group description.

      Since $f$ is a continuous map, it induces a map $f^* : H_k (S_n) \to H_k(S_n)$. For $k = n$, we have $H_n(S_n) = \Z$. We call $n = f^*(1)$ the degree of $f$.

    • Give an example of a smooth map $\R \to \R^2$ which is an immersion but not an embedding.

      The map $F(\theta) = (\cos \theta, \sin \theta)$ from $\R$ into $\R^2$ is an immersion because the dimension of the tangent space is preserved at every point. It's not an embedding because $2\pi \Z$ go to the same point.
      Let $F : \R \to \R^2$ be the map $F(\theta) = (\cos \theta, \sin \theta)$. Clearly $F$ is smooth, so it remains to check that $dF$ is injective. Computation gives \[ dF = \nabla F = \begin{bmatrix} -\sin \theta \cr \cos \theta \end{bmatrix} \] This map $dF$ has full rank if and only if it is not zero. Since $\sin$ and $\cos$ are phase shifts of each other, they have no common roots. Thus $dF$ has full rank for every $\theta$. We have shown that $F$ is an immersion. But since $F(2\pi \Z) = (1, 0)$ we see that $F$ is not an embedding.

    • Calculate the Lie brack of the following vector fields on $\R^3$: \[ \begin{aligned} \partial_x + y \partial_y + 2(x + y^2) \partial_z \cr \partial_y + 2y \partial_z \end{aligned} \] Do there exist integral surfaces for this vector field?       You should read what is the Lie bracket and what does the Frobenius theorem say. For this problem, we are looking for a two dimensional surface that has tangent vectors $X = \partial_x + y \partial_y + 2(x + y^2) \partial_z, Y = \partial_y + 2y \partial_z$ at every point. As described in the above article on the Frobenius theorem, this means we need $[X_p, Y_p] \in \textrm{span } (X_p, Y_p)$. Computation in coordinates via \[ [X, Y] = (X \, Y^i - Y \, X^i)\partial_{x^i} \] gives \[ [X, Y] = -Y \] which is in the span of $X, Y$. Since $X, Y$ always form a basis of $\R^2$, they form a tangent distribution (see tangent distribution), in fact an involutive distrubtion. By Frobenius' theorem, the involutive distributions precisely characterize the completely integrable surfaces. Hence, the answer to the original question is yes.
      Let $M$ be a smooth manifold and $D$ a subset of the tangent bundle $TM$ with corresponding projection map $\pi : E \to M$. Then we call $D$ a tangent distribution if $D$ satisfies the following
      1. $D$ is a submanifold of $TM$
      2. For $p \in M$, then $\pi^{-1}(p)$ is a subspace of $T_p M$
      3. $D$ is a smooth vector bundle over $M$
      We say that a tangent distribution is involutive if for any sections $X, Y$ (vector fields) of $D$, their Lie bracket $[X,Y]$ is also a section of $D$.

      Since $X = \partial_x + y \partial_y + 2(x + y^2) \partial_z, Y = \partial_y + 2y \partial_z$ form a basis for an affine two-dimensional subspace at every point, they describe a tangent distribution $D$. To see that an integral surface exists, by Frobenius' theorem it suffices to check that $D$ is involutive.

      In coordinates the Lie bracket is given by \[ [X, Y] = (X \, Y^i - Y \, X^i)\partial_{x^i} = X \, Y^i \partial_{x^i} - Y \, X^i \partial_{x^i}. \] Our computation is thus \[ \begin{aligned} [X,Y] &= (X(0)\partial_x + X(1) \partial_y + X(2y)\partial_z) - (Y(1)\partial_X + Y(y)\partial_y + Y(2(x + y^2))) \partial_z \cr &= 0 + 0 + 2y\partial_z - (0 + \partial_y + 4y\partial_z) \cr &= - \partial_y - 2y\partial_z \cr &= -Y \end{aligned} \] Hence, $D$ is involutive so there exists an integral surface.

      For the second (non-trivial example), consider the differential form \[ \omega = dx \wedge dy + dz \wedge dt \] and note that \[ \begin{aligned} \omega \wedge \omega &= (dx \wedge dy + dz \wedge dt) \wedge (dx \wedge dy + dz \wedge dt) &= 0 + (dx \wedge dy \wedge dz \wedge dt) + (dz \wedge dt \wedge dx \wedge dy) + 0 &= 2dx \wedge dy \wedge dz \wedge dt \end{aligned} \] where the last step followed since we preformed an even permutation on the elements in the term $dz \wedge dt \wedge dx \wedge dy$ to rewrite it as $dx \wedge dy \wedge dz \wedge dt$.

      The space has one hole surrounding the $x$-axis. We can deformation retract the entire space onto $S^1$ as follows

      We claim that we can deformation retract the space $\R^3 \backslash \{ x, 0, 0 \}$ to $S^1$ via the following map. First, project a every point onto the $y-z$ plane via $\pi(x, y, z) = (0, y, z)$. Then, map each point to $S^1$ via $p(0, y, z) = (y, z) / (y^2 + z^2)$.
