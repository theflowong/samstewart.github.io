  3. Read about the Decomposition of spectrum and understand the example

Interesting ideas during the session

  1. Right and left shift operators have the same spectrum, but the right shift operator has no eigenvalues
  2. Group structure allows one to reduce local problems at a group element $g$ to local problems at $1$ by treating $g$ as the identity. This is equivalent to transforming the problem to the origin.
  3. Invertability of a linear operator is stable precisely because $(1 + r)$ is invertable when $\abs{r} < 1$.
  4. Stability of inverses means that resolvant set is open, so spectrum is closed
  5. Boundary points of the spectrum are never eigenvalues (?)
  6. Unlike finite dimensional case, there are two ways in infinite dimensional spaces for a linear operator to be non-invertable: it can fail to be onto, or it can fail to be injective. Exactly how badly it fails to be onto divides the spectrum into several pieces.
