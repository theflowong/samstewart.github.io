Part A

    • Define what it mean for two paths $\alpha, \beta : [0, 1] \to X$, with the same start points and the same end points, to be homotopic. Show that this is an equivalence relation.

      See what is the fundamental group?.

      Let $\alpha, \beta : [0, 1] \to X$ such that $\alpha(0) = \beta(0) = x_0 \in X$ be loops in $X$. We say that $\alpha$ is homotopic to $\beta$ if there exists a function $H : [0, 1] \times [0, 1] \to X$ such that the following hold
      1. H(0, \cdot) = \alpha
      2. H(1, \cdot) = \beta
      3. $H$ is continuous
      We sketch the proof that this is an equivalence relation

      • Reflexivity

        Simply consider the identity homotopy $H(t, x) = \alpha(x)$.

      • Symmetry

        Let $H$ be the homotopy between $\alpha$ and $\beta$. Then $\widetilde{H} = H(1 - t, x)$ is a homotopy between $\beta$ and $\alpha$ since $1 - t$ is a continuous function.

      • Transitivity

        Let $H_1$ be the homotopy between $\alpha$ and $\beta$, and $H_2$ the homotopy between $\beta$ and $\gamma$. Then $\widetilde{H}(x, t) = H_1$ for $t \in [0, 1/2]$ and $\widetilde{H}(x, t) = H_2$ for $t \in (1/2, 1]$ is a homotopy between $\alpha$ and $\gamma$.

    • Define what it means for a space to be simply connected.

      Simply connected means the space has no holes. See what is the fundamental group?.
      If $X$ is a path connected space, we say that $X$ is simply connected if \[ \pi_1(X, x_0) = 1. \] That is, the fundamental group of $X$ is trivial.

    • Give a complete statement of the Seifert-van Kampen theorem relating the fundamental groups of $U, V, U \cup V$, and $U \cap V$, including all necessary assumptions.

      See What does Van Kampen's Theorem Really Say
      Let $X = U \cup V$ be a path-connected space with $U, V$ open path-connected subsets such that $U \cap V$ is path connected. Then the fundamental group of $X$ is given by \[ \pi_1(X, x_0) = \pi_1(U, u_0) * \pi_1(V, v_0) / N \] where $*$ denotes the free product and $N$ is generated by the set \[ \{ \pi_1(i_U)(\gamma)\pi_1(i_V)(\gamma)^{-1} \mid \gamma in \pi_1(U \cap V, y_0) \}. \] where $i_U, i_V$ are the inclusion maps from $U, V$ to $X$, respectively.

    • Describe the fundamental group of the Klein bottle using generators and relations.

      Decompose $\R^2$ into two halves $U, V$ with an overlapping strip in the middle. Compute the fundamental group of both halves via a circle retract and see that both halves have fundamental group $\Z * \Z$. By Van Kampen's theorem, we have that \[ \pi_1(\R^2 \backslash \{ (-1, 0), (1, 0) \}, 0) = \pi_1(U, (-1, 0)) * \pi_1(V, (1, 0)) = \Z * \Z \]. We don't need to worry about the quotient group since the intersection $U \cap V$ is simply connected.
      The fundamental group is \[ \pi_1(\R^2 \backslash \{ (-1, 0), (1, 0) \}, 0) = \Z * \Z. \]
      Let $U = \{ (x, y) \in \R^2 \mid x < 1 \}$ and $V = \{ (x, y) \in \R^2 \mid x > -1 \}$. Then $U \cap V = \{ (x, y) \mid x \in [-1, 1] \}$. One can show that $U$ and $V$ are homeomorphic to $\R^2 \backslash \{ (0, 0) \}$. By constructing a deformation retract from $\R^2 \backslash \{ (0, 0) \}$ to $S^1$, we can see that \[ \pi_1($\R^2 \backslash \{ (0, 0) \}$) = \pi_1(S^1) = \Z. \] Thus, $\pi_1(U) = \pi_1(V) = \Z$. Since $U \cap V$ is a convex set in $\R^2$, it is simply connected. Hence, in the statement of Van Kampen's Theorem, we have $N = 1$. Hence we can conclude from Van Kampen's Theorem that \[ \pi_1($\R^2 \backslash \{ (0, 0) \}$) = \pi_1(U) * \pi_1(V) / N = \Z* \Z / 1 = \Z * \Z. \]

    • Calculate the homology groups of the torus $S^1 \times S^1$.

      See what is homology?. Build the torus from one $0$-simplex, three $1$-simplices, and two $2$-simplices as follows The first homology group $H_0 = \Z$ since the space is connected. Since the torus is path connected, then $H_1 = \pi_1 = \Z \times \Z = \Z \oplus \Z$. Finally, we have $H_2 = \langle \lambda_1 - \lambda_2 \rangle / \langle 0 \rangle = \Z$. Hence, the homology is \[ H_k(T) = \begin{cases} \Z & \textrm{ if } k = 0, 2 \cr \Z \oplus \Z & \textrm { if } k = 1 \end{cases} \]
      From first definitions alone we can compute the homology group. We start by computing the kernels and images of the boundary maps. \[ \begin{aligned} \textrm{ker } \partial_0 &= \Z \cr \textrm{img } \partial_0 &= 0 \cr \textrm{ker } \partial_1 &= \langle a, b, c \rangle \cr \textrm{img } \partial_1 &= \langle 0 \rangle \cr \textrm{ker } \partial_2 &= \langle \lambda_1 - \lambda_2 \rangle \cr \textrm{img } \partial_2 &= \langle a + b - c\rangle \cr \textrm{ker } \partial_3 &= 0 \cr \textrm{img } \partial_3 &= 0\rangle \end{aligned} \] Note that $\textrm{ker } \partial_2$ because \[ \partial_2(p \lambda_1 + q \lambda_2) = (p + q)(a + b - c) = 0. \] Since $a + b - c$ is a element of the basis $\{ a, b, a + b - c \}$ for $C_1$, then $p + q = 0$ so $p = -q$. With this information we can compute the homology groups as follows \[ \begin{aligned} H_0 &= \frac{\Z}{0} = \Z \cr H_1 &= \frac{\textrm{ker } \partial_1}{\textrm{img } \partial_2} = \frac{\langle a, b, c \rangle}{\langle a + b - c \rangle} = \Z \oplus \Z \oplus \Z / \Z = \Z \oplus \Z \cr H_2 &= \frac{\textrm{ker } \partial_2}{\textrm{img } \partial_3} = \frac{\langle \lambda_1 - \lambda_2 \rangle}{0} = \Z \end{aligned} \] Thus, the homology of the torus is \[ H_k(T) = \begin{cases} \Z & \textrm{ if } k = 0, 2 \cr \Z \oplus \Z & \textrm { if } k = 1 \end{cases} \]

    • For $n > 0$, define the degree of a continuous map $S^n \to S^n$.

      The general definition (stated below) generalizes the simple idea that the degree of $f$ measures how many times it winds a loop of $S^1$ around $S^1$.

      For example, the degree of the identity is $1$, the degree of $z \mapsto z^2$ is two, the degree of a reflection is $-1$, and the degree of the antipodal map is $1$ (a rotation by $pi$, so homotopic to the identity).

      Most importantly, the maps $f$ and $g$ are homotopic if and only if they have the same degree. Hence, we can see that the fundamental group of $S^1$ is indeed $\Z$. This construction is earlier than the fundamental group description.

      Since $f$ is a continuous map, it induces a map $f^* : H_k (S_n) \to H_k(S_n)$. For $k = n$, we have $H_n(S_n) = \Z$. We call $n = f^*(1)$ the degree of $f$.

    • Define $f : \R^3 \to \R$ by \[ f(x,y,z) = x^2 + y^2 - z^4 + 2z^2 \] Find all the singular points and singular value of $f$.

      By definition, we are looking for points $p$, such that $df_p = \nabla f(p) = 0$. Computation gives $df = \nabla f = (2x, 2y, 4z(1 - z^2))$ with $\nabla f = 0$ for critical point \[ (0, 0,0), \; (0, 0, 1), \; (0, 0, -1). \] These give critical values $0, 1, 1$ respectively. In agreement with Sards' theorem, the measure of the set of critical values is zero.

    • Suppose we have a smooth map $f : \R \to \R^{n + 1}$, of the form \[ f(t) = (x_0(t), \ldots, x_n(t)). \] Let $p$ be the projection from $\R^{n + 1}$ to $\R^n$ defined by $p(x_0, \ldots, x_n) = (x_1, \ldots, x_n)$. Give precise conditions on $f$ so that $f$ is an immersion but $p \circ f$ is not.

      The problem is that a nonzero tangent vector in $\R^2$ might have zero components, that is project downward to a zero vector in $\R$. As an example, consider the map \[ f(t) = (\cos t, \sin t) \]

      Notice that the tangent vector does not vanish. On the other hand, the projection \[ p \circ f = \sin t \] has places with zero tangent vector which correspond to points where the tangent vector on $S^1$ is either horizontal or vertical At the points where the tangent vector is zero, the function is not an immersion. Hence, the right condition is that the gradient of $f$ does not vanish while the gradient of $(x_1, \ldots, x_n)$ does vanish.

      Assume that \[ df = \nabla f = \begin{bmatrix} x_0' \cr x_1' \cr \vdots \cr x_n' \end{bmatrix} \neq 0 \] and that there exists a point $t_0$ such that \[ d(f \circ p) = \begin{bmatrix} x_1'(t_0) \cr \vdots \cr x_n'(t_0) \end{bmatrix} = 0. \] By definition $df$ is injective since it is nonzero, yet $d(f \circ p)$ is not injective at the point $t_0$. Hence, $f$ is an immersion but $p$ is not.

    • Calculate the Lie brack of the following vector fields on $\R^3$: \[ \begin{aligned} \partial_x + y \partial_y + 2(x + y^2) \partial_z \cr \partial_y + 2y \partial_z \end{aligned} \] Do there exist integral surfaces for this vector field? (In other words, for any point $p$, does there exist a smooth surface $S$ in $\R^3$ containing $p$ whose tangent space at any point $q \in S$ is spanned by these vector fields?)

      You should read what is the Lie bracket and what does the Frobenius theorem say. For this problem, we are looking for a two dimensional surface that has tangent vectors $X = \partial_x + y \partial_y + 2(x + y^2) \partial_z, Y = \partial_y + 2y \partial_z$ at every point. As described in the above article on the Frobenius theorem, this means we need $[X_p, Y_p] \elem \textrm{span } (X_p, Y_p)$. Computation in coordinates via \[ [X, Y] = (X \, Y^i - Y \, X^i)\partial_{x^i} \] gives \[ [X, Y] = -Y \] which is in the span of $X, Y$. Since $X, Y$ always form a basis of $\R^2$, they form a tangent distribution (see tangent distribution), in fact an involutive distrubtion. By Frobenius' theorem, the involutive distributions precisely characterize the completely integrable surfaces. Hence, the answer to the original question is yes.
      Let $M$ be a smooth manifold and $D$ a subset of the tangent bundle $TM$ with corresponding projection map $\pi : E \to M$. Then we call $D$ a tangent distribution if $D$ satisfies the following
      1. $D$ is a submanifold of $TM$
      2. For $p \in M$, then $\pi^{-1}(p)$ is a subspace of $T_p M$
      3. $D$ is a smooth vector bundle over $M$
      We say that a tangent distribution is involutive if for any sections $X, Y$ (vector fields) of $D$, their Lie bracket $[X,Y]$ is also a section of $D$.

      Since $X = \partial_x + y \partial_y + 2(x + y^2) \partial_z, Y = \partial_y + 2y \partial_z$ form a basis for an affine two-dimensional subspace at every point, they describe a tangent distribution $D$. To see that an integral surface exists, by Frobenius' theorem it suffices to check that $D$ is involutive.

      In coordinates the Lie bracket is given by \[ [X, Y] = (X \, Y^i - Y \, X^i)\partial_{x^i} = X \, Y^i \partial_{x^i} - Y \, X^i \partial_{x^i}. \] Our computation is thus \[ \begin{aligned} [X,Y] &= (X(0)\partial_x + X(1) \partial_y + X(2y)\partial_z) - (Y(1)\partial_X + Y(y)\partial_y + Y(2(x + y^2))) \partial_z \cr &= 0 + 0 + 2y\partial_z - (0 + \partial_y + 4y\partial_z) \cr &= - \partial_y - 2y\partial_z \cr &= -Y \] Hence, $D$ is involutive so there exists an integral surface.

    • Let $M = \R^3 \backslash \{ (x, 0, 0) \mid x \in \R \}$. For which values of $k$ does there exist a smooth $k$-form $\omega$ on $M$ such that $d\omega = 0$, but $\omega \neq d\tau$ for any $(k - 1)$-form $\tau$?

      The space has one hole surrounding the $x$-axis. We can deformation retract the entire space onto $S^1$ as follows

      We claim that we can deformation retract the space $\R^3 \backslash \{ x, 0, 0 \}$ to $S^1$ via the following map. First, project a every point onto the $y-z$ plane via $\pi(x, y, z) = (0, y, z)$. Then, map each point to $S^1$ via $p(0, y, z) = (y, z) / (y^2 + z^2)$. Clearly both maps are continuous and leave $S^1$ fixed so their composition is continuous and leaves $S^1$ fixed.

      Since cohomology is a topological invariant, we have shown that because $\R^3 \backslash \{ x, 0, 0 \}$ is homotopy equivalent to $S^1$, then \[ H^n_{dR}($\R^3 \backslash \{ x, 0, 0 \})$ = H^n_{dR}(S^1). \] We know the homology of the circle is \[ H^n_{dR}(S^1) = \begin{cases} \R & \text{ if } n = 0, 1 \cr 0 \text{ otherwise } \end{cases} \] so non-exact $k$-forms exist precisely for $k = 0, 1$.
